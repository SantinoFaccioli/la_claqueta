## Temática del TPE
]**Reseñas de Películas**

## Descripción de la temática
El sitio consiste en una plataforma dinámica con Server Side Rendering que permite la visualización y administración de un catálogo de reseñas de películas.Los usuarios pueden explorar las películas organizadas por categorías (como Acción, Drama, Comedia, etc.).El administrador del sitio tiene permisos exclusivos para crear, editar y eliminar tanto las películas como las categorías.

## Modelo de Datos
El modelo se basa en una relación de **1 a N**, donde una categoría puede agrupar múltiples películas, pero cada película pertenece a una única categoría
